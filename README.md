# 🏦 Guru99 Bank Automation Framework

## 📌 Project Overview

This project is an end-to-end Selenium Automation Framework developed using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM).

The framework automates major banking functionalities of the Guru99 Bank Demo application and follows industry-standard automation practices such as Data-Driven Testing using XML, TestNG Listeners, Screenshot Capture on Failure, and Extent Reports.

---

## 🚀 Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager
- Page Object Model (POM)
- XML Data Driven Testing
- TestNG Listeners
- Extent Reports
